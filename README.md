## E-Commerce Sales Dashboard

## 📌 Project Overview
This project analyzes e-commerce sales data using **Python**, **SQL**, and **Power BI** to uncover insights related to revenue, orders, product performance, and delivery status.


## 🛠 Tools & Technologies
- Python (pandas)
- SQL
- Power BI
- GitHub


## 📂 Dataset
- Total Records: 500 orders  
- Data includes customer details, product category, quantity, revenue, order status, and dates.


## 📊 Key Insights
- **Total Revenue:** 709K  
- **Total Orders:** 500  
- **Average Order Value:** 2.12K  
- **Delivery Rate:** 67%


## 📈 Dashboard Features
- KPI cards for Revenue, Orders, AOV, and Delivery Rate
- Monthly revenue trend (line chart)
- Revenue by category (bar chart)
- Order status breakdown (donut chart)
- Top products by sales (table)
